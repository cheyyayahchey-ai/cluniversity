# CL University - School Management System (C# Blazor)

Full-stack C# school management system with QR payment.

## Features
- Student, Lecturer, Staff Management
- Department & Course Management
- Bakong KHQR Payment with:
  - QR Code only display
  - 4-minute countdown timer
  - Reload QR button
  - Auto receipt & print on payment success
- Salary/Payroll Management
- Dashboard with Statistics
- USD/KHR currency conversion

## Run the Application

```bash
cd CLUniversity
dotnet restore
dotnet run
```

Then open: https://localhost:5001

## Requirements
- .NET 8.0 SDK
